# IPL Victory Probability Estimator  
A machine learning-based project to estimate the probability of victory in an IPL match. This tool provides real-time predictions through a user-friendly Streamlit app, helping cricket enthusiasts analyze match scenarios.

## **Features**  
- Predict the probability of victory for an IPL team based on match conditions.  
- Input parameters include:  
  - Batting team  
  - Bowling team  
  - Host city  
  - Target score  
  - Current score  
  - Overs done  
  - Wickets fallen  
- Interactive and visually appealing results displayed in a Streamlit app.  

## **Technologies Used**  
- **Python**: For data preprocessing, model development, and prediction.  
- **Pandas & NumPy**: For data manipulation and analysis.  
- **Scikit-learn**: For machine learning model development.  
- **Streamlit**: For creating an interactive web application.
